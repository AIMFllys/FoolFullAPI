## 聊天消息发送流程

```
用户点击发送
  │
  ├── 1. 立即显示用户消息（乐观更新）
  ├── 2. 显示"AI 正在思考..."动画
  ├── 3. 发送 POST /api/chat
  ├── 4. 收到响应后，根据 step 等待不同时长：
  │      Step 1: 3~5秒  Step 2: 2~3秒  Step 3: 1~2秒
  │      Step 4: 1~2秒  Step 5: 4~6秒  Step 6: 0秒
  ├── 5. 隐藏思考动画
  ├── 6. 打字机效果逐字显示回复
  ├── 7. 更新剩余次数
  └── 8. Step 6 → 触发彩蛋动画
```

---

## 打字机效果

```typescript
const typewriterEffect = (text: string, onChar: (partial: string) => void) => {
  let i = 0;
  const interval = setInterval(() => {
    i++;
    onChar(text.slice(0, i));
    if (i >= text.length) clearInterval(interval);
  }, 30 + Math.random() * 20); // 每 30~50ms 一个字符
};
```

---

## Step 2 进度条特殊渲染

Step 2 的回复包含进度条格式，前端需要特殊处理：
- 逐行显示（每行间隔 1 秒）
- ✅ 图标带绿色闪烁动画
- ⏳ 图标带旋转动画
- 百分比数字带递增动画（从 0% 到 67%）

---

## Step 5 中断效果

Step 5 的回复包含 `⚠️ [系统提示]` 标记，前端检测到后：
- 正常内容用打字机效果显示
- 遇到 `⚠️` 时暂停 1 秒
- 然后用红色/橙色样式显示系统提示
- 消息气泡底部添加红色闪烁边框

---

## Step 6 彩蛋动画

- 消息气泡使用彩虹渐变背景
- 页面顶部飘落彩带/纸屑动画（canvas 或 CSS animation）
- 🎉 表情放大弹跳动画
- 可选：播放欢快音效
