## 测试计划

### 测试类型

| 类型 | 范围 | 工具 |
|------|------|------|
| 单元测试 | 工具函数、验证器、模板生成 | Vitest |
| 集成测试 | API 路由、DB 操作 | Vitest + supertest |
| E2E 测试 | 完整用户流程 | 手动测试 |
| 安全测试 | 注入、XSS、提示词注入 | 手动测试 |
| 压力测试 | 并发请求 | autocannon |
