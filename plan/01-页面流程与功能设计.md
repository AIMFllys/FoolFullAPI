## 各页面详细功能

### 1. 落地页（Landing Page）

**路由**：`/`

**功能要素**：
- 顶部导航栏：Logo（"Claude Opos4 母4.6"）、"登录"按钮、"服务条款"链接、"隐私政策"链接
- Hero 区域：
  - 大标题："Claude Opos4 母4.6 – 地表最强对话式 AI"
  - 副标题："超越一切已知 AI 模型，为你而来"
  - CTA 按钮："立即免费体验" → 跳转到注册/登录页
- 特性展示区（3~4 个卡片）：
  - "超强推理能力" — 图标 + 简短描述
  - "百万级上下文窗口" — 图标 + 简短描述
  - "多模态理解" — 图标 + 简短描述
  - "极速响应" — 图标 + 简短描述
- 底部 Footer：版权信息、服务条款链接、隐私政策链接

**设计风格**：
- 深色主题（模仿高端 AI 产品风格）
- 渐变背景（深蓝 → 紫色）
- 科技感字体和动画效果
- 响应式布局，适配移动端

---

### 2. 注册/登录页（Auth Page）

**路由**：`/auth`

**注册表单**：
- 用户名输入框（4~20 字符，字母数字下划线）
- 密码输入框（6~32 字符）
- 确认密码输入框
- 勾选框："我已阅读并同意《服务条款》和《隐私政策》"（必选）
- 注册按钮

**登录表单**：
- 用户名输入框
- 密码输入框
- 登录按钮

**验证规则**：
- 前端实时验证（格式、长度、密码一致性）
- 后端验证（用户名唯一性、密码正确性）
- 不需要邮箱，仅用户名 + 密码
- 密码后端使用 bcrypt 哈希存储
- 登录成功后返回 JWT Token

---

### 3. 功能选择页（Select Page）

**路由**：`/select`（需登录）

**功能要素**：
- 欢迎语："欢迎回来，{username}！"
- 两个大卡片选择：
  - **在线对话**：图标 + "直接在浏览器中与 AI 对话" + "开始对话"按钮 → `/chat`
  - **API 接入**：图标 + "获取 API Key，接入你的 IDE 或应用" + "查看文档"按钮 → `/api-docs`
- 用户状态显示：
  - 剩余交互次数：`{18 - ask_count} / 18`
  - 如果已封禁，显示"您的体验次数已用完"

---

### 4. 聊天界面（Chat Page）

**路由**：`/chat`（需登录）

**功能要素**：
- 顶部栏：模型名称 "Claude Opos4 母4.6"、返回按钮、剩余次数
- 消息区域：
  - 用户消息（右侧气泡，蓝色）
  - AI 回复（左侧气泡，灰色）
  - 支持 Markdown 渲染（代码块、加粗等）
  - 自动滚动到最新消息
- 输入区域：
  - 多行文本输入框
  - 发送按钮（Enter 发送，Shift+Enter 换行）
  - 发送中状态：显示"AI 正在思考..."动画

**特殊状态处理**：
- 封禁状态：输入框禁用，显示"体验次数已用完"
- 每日上限：显示"今日体验次数已用完，请明日再来"
- Step 6 彩蛋：显示特殊的愚人节彩蛋样式（彩色背景、🎉 表情、彩带飘落）

**交互细节**：
- 每次发送消息后，前端显示 2~4 秒的"思考中"动画（Step 1 和 Step 5 延迟更长，模拟深度思考）
- AI 回复使用打字机效果逐字显示
- Step 2 的进度条使用特殊的动画渲染（逐步出现 ✅）
- Step 6 的彩蛋消息使用彩带飘落动画

---

### 5. API 文档页（API Docs Page）

**路由**：`/api-docs`（需登录）

**功能要素**：
- API Key 展示区：显示 / 复制 / 重新生成
- API 使用说明：
  - 端点：`POST /api/v1/chat`
  - 请求头：`Authorization: Bearer {API_KEY}`
  - 请求体：`{ "message": "你的问题" }`
  - 响应体：`{ "reply": "...", "step": 1, "remaining": 17 }`
- 代码示例：cURL / Python / JavaScript
- 注意事项：每个账号最多 18 次调用，API Key 仅在活动期间有效
