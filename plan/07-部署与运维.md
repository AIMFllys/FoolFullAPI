## 部署步骤

### 1. 服务器准备

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs nginx
npm install -g pm2
```

### 2. 项目构建

```bash
git clone <repo-url> /var/www/april-fools-ai
cd /var/www/april-fools-ai

cd client && npm install && npm run build
cd ../server && npm install && npm run build
```

### 3. 环境变量

```bash
cd /var/www/april-fools-ai/server
cp .env.example .env
nano .env  # 填入实际配置
```

### 4. 启动后端

```bash
pm2 start dist/index.js --name april-fools-api
pm2 save && pm2 startup
```

### 5. Nginx 配置

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        root /var/www/april-fools-ai/client/dist;
        try_files $uri $uri/ /index.html;
    }

    location /api/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_cache_bypass $http_upgrade;
    }
}
```

### 6. SSL 证书

```bash
sudo apt-get install certbot python3-certbot-nginx
sudo certbot --nginx -d your-domain.com
```

## 环境变量配置（.env.example）

```bash
# ===== 服务器 =====
PORT=3000
NODE_ENV=production

# ===== 前端 =====
FRONTEND_URL=http://localhost:5173

# ===== JWT =====
JWT_SECRET=your-super-secret-jwt-key-at-least-32-chars-long

# ===== 数据库 =====
DB_PATH=./data/database.db

# ===== 七牛云 AI API =====
QINIU_API_URL=https://api.qnaigc.com/v1/chat/completions
QINIU_API_KEY=your-qiniucloud-api-key
KIMI_MODEL=kimi                    # Kimi 模型标识（Step 1, 5）
DEEPSEEK_MODEL=deepseek-chat       # DeepSeek 模型标识（Step 3, 4）

# ===== 额度限制 =====
MAX_ASK_PER_USER=18                # 每用户最大交互次数（6步 × 3轮）
MAX_DAILY_API_CALLS=1000           # 全局每日 API 调用上限
IP_BAN_THRESHOLD=50                # IP 封禁阈值
IP_BAN_DURATION_MINUTES=30         # IP 封禁时长

# ===== 活动 =====
EVENT_END_DATE=2026-04-02T00:00:00+08:00
DATA_CLEANUP_ENABLED=true
```

### 环境变量验证

## 数据清理策略（4月2日）

### 定时任务

```typescript
import cron from 'node-cron';

// 每日 00:00 重置全局 API 计数
cron.schedule('0 0 * * *', () => {
  db.prepare('UPDATE global_stats SET daily_api_count = 0').run();
});

// 4月2日 00:00 清理用户数据
cron.schedule('0 0 2 4 *', () => {
  if (new Date().getFullYear() === 2026) {
    fs.copyFileSync('./data/database.db', `./data/backup_${Date.now()}.db`);
    db.prepare('DELETE FROM chat_logs').run();
    db.prepare('DELETE FROM users').run();
    console.log('[CRON] 用户数据已清理');
  }
});
```

### 清理范围

| 数据 | 操作 |
|------|------|
| users 表 | DELETE ALL |
| chat_logs 表 | DELETE ALL |
| global_stats 表 | 保留 |
| 表结构 | 保留 |

### 清理后行为

所有 API 路由（除 health/stats）返回 410：
```json
{ "success": false, "error": { "code": "EVENT_ENDED", "message": "愚人节活动已结束，感谢参与！" } }
```
